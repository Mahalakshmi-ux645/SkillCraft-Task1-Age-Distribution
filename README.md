# SkillCraft Task 1 - Age Distribution Visualization
This project visualizes age distribution in a population using Python, Pandas, and Matplotlib. Developed as part of SkillCraft Technology Data Science Internship Task 01, it includes graphical analysis through histograms and population distribution charts to understand demographic patterns.

## Overview

This project was completed as part of the SkillCraft Technology Data Science Internship Task 01. The objective of this task is to visualize the distribution of a continuous variable using graphical representations.

## Dataset

The dataset contains age and population data used to analyze the distribution of age groups within a population.

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Google Colab

## Project Steps

1. Created and prepared the dataset.
2. Loaded the dataset using Pandas.
3. Performed basic data analysis.
4. Visualized age distribution using charts.
5. Generated graphical insights using Matplotlib.

## Visualizations

* Age Distribution Graph
* Histogram of Age Distribution
